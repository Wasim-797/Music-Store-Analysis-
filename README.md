

# 🎵 SQL Music Store Analysis

This SQL project focuses on analyzing a Music Store database to uncover valuable business insights. By exploring customer behavior, sales trends, popular genres, top artists, and revenue patterns through SQL queries, this project demonstrates how data analysis can support smarter business decisions and drive growth. It is an excellent hands-on project for strengthening SQL skills and understanding real-world data analysis workflows.



---

## 🛠️ Tools Used
- PostgreSQL
- PgAdmin4
- SQL

---

## 📊 Key Analysis
- Top customers by spending
- Best-selling genres & artists
- Revenue analysis by country/city
- Popular tracks analysis
- Customer purchase behavior

---

## 💡 SQL Concepts
- Joins
- Aggregate Functions
- Subqueries
- CTEs
- Window Functions
